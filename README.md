# 🛍️ Avanti Store - Teste Técnico

Projeto desenvolvido como teste técnico para a Avanti, com foco em fidelidade visual (pixel-perfect) a partir de um layout no Figma.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Swiper.js (utilizado para o carrossel de produtos)

---

## 🎯 Objetivo

Reproduzir o layout proposto com máxima precisão, respeitando:

* Tipografia
* Espaçamentos
* Cores
* Componentes visuais
* Responsividade

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem responsiva, garantindo adaptação para:

* 📱 Mobile
* 💻 Desktop

---

## ⚙️ Funcionalidades

* Campo de busca com exibição do termo digitado
* Menu responsivo (mobile)
* Carrossel de produtos com Swiper.js
* Layout adaptado conforme o design do Figma
